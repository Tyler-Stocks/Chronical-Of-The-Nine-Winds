# Text-Based-RPG

This is a Text-Based RPG that I am creating in Python for my final project for my IT class. The program is (Mostly) object oriented in its nature. I decided that this was a good idea because it makes it easier to call the classes, than the call the same functions over and over again. I hope to someday make this have a UI using Django.
I would also like to optimize my program to be more efficient. Finnaly, I would like to package all of my files into an EXE so that it can be easily run.
