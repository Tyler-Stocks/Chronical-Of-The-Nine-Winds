Hello, here are some quick guidelines for naming things in the project.

1. Folder, or file names should never have spaces, I have been having a difficult time making this type of file or folder work so it is best just to avoid.
2. Variable Names should never be capital, this is pretty self explanitory, just thought I'd include it just in case.
3. Class names should have each word start with a capital.
4. File, and function names should use snake_case.

I May add some other rules here later, however this is all for now.
